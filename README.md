# 🚀 Sentiment Analysis Web Application

## 📌 Overview
This project is a Machine Learning based Sentiment Analysis Web Application that classifies user input text into Positive 😊 or Negative 😡 sentiment using NLP techniques and Flask.

## 🎯 Objective
- Analyze text sentiment
- Build ML model
- Deploy as web app

## 🛠️ Technologies Used
- Python
- Flask
- Scikit-learn
- HTML, CSS

## 📁 Project Structure
sentiment-analysis/
│
├── app.py
├── sentiment_model.pkl
├── requirements.txt
├── templates/
│   └── index.html

## 🚀 Features
- Real-time sentiment prediction
- Simple UI
- Fast processing

## ▶️ How to Run
1. Install:
pip install -r requirements.txt

2. Run:
python app.py

3. Open:
http://127.0.0.1:5000/
